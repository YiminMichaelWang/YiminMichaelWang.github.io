Research portfolio

Interpretable, robust, and data-aware machine learning.

My work connects algorithm design with scientific applications: Shapley-value attribution for LLMs, evolutionary optimization for biological discovery, and data-centric strategies for noisy real-world datasets.

May 2025 - Present Interpretable and Robust AI Research Group Advisor: Dr. Fan Yang

Efficient Shapley Estimation across Tabular, Image, and Graph Data in LLMs

This project develops a unified framework for estimating Shapley values efficiently across several data modalities. The framework uses data-type priors to reduce variance and cost while preserving black-box compatibility.

  • Designed an attribution framework with mathematical proofs of unbiasedness and faithfulness.
  • Used antithetic grouping for tabular data, random cropping for images, and single-hop sampling for graphs.
  • Extended dataset-level valuation to GPT-2 small for data Shapley in LLM training.
  • Performed pixel-level and frame-level Shapley analysis for image and video sensitivity reports.
  • Studied functional subgraphs in LLMs that contribute most to target outputs without gradients or retraining.
  • Extended Shapley attribution to token-level explanations for long-context prompts.
May 2023 - August 2025 DataMine Research Group Advisor: Dr. Natalia Khuri

Data-Centric Artificial Intelligence for Environmental Toxicology

This line of work studies how data selection and data quality interventions affect model performance in environmental toxicology and biomedical datasets, in collaboration with Wake Forest University School of Medicine.

  • Evaluated data-centric approaches, including coreset methods, using paired t-tests and model performance comparisons.
  • Implemented and tested genetic algorithms for dataset coreset selection.
  • Ran parallelized grid searches for genetic algorithm hyperparameters.
  • Visualized selection behavior, minority ratios, and runtime using seaborn and matplotlib.
2023 - 2025 Computational biology + optimization Python · R · XGBoost · GA

Multi-Objective Optimization for scRNA-seq and QSAR Modeling

I contributed to methods and tooling for high-dimensional biological data, including clustering, cell-type prediction, and compound discovery with multi-objective evolutionary optimization.

  • Contributed to the scrnabench GitHub package for scRNA-seq benchmarking.
  • Implemented a multi-objective genetic algorithm for clustering noisy high-dimensional datasets.
  • Built R scripts to retrieve, annotate, clean, and prepare scRNA-seq data for XGBoost cell-type prediction.
  • Conducted experiments across chemical datasets using coreset selection methods and classification models.
